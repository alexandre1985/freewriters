Book Templates

What is a book template?

A LaTeX book template is a easy way to edit a structure of a LaTeX book, so that you don't have to know advanced skills with LaTeX in order to write one. All you need to do is download, unzip, and edit the .tex file.

Book Templates

Ebook

This ebook template is for those wishing to create a book in a format that is easily readable on electronic screens. To facilitate this, the template features a layout with narrow margins and a large font size to use as much of your screen real estate as possible.
Book content can be divided by parts, chapters, sections, subsections and subsubsections.
